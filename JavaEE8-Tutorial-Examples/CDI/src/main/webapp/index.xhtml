The CDI examples:

  • Simple Greeting: Very simple CDI example. It uses a page template.

  • GuessNumber-CDI: Guess the server random number. The hint dynamically changes based on your input.

  • ToDo List: This example demos producer field. The App: Create a ToDo item and save in DB.(need to run DB to install/run this app).

  • Encoder: This example demos producer methods. The App: Encode a input String.

  • Encoder2: This example demos injecting a bean as "alternative". The App: Same as Encoder

  • Encoder3: This example demos injecting a bean as "decorator". The App: Same as Encoder
