The JSF examples:

  • Greeting: This is the Hello World application of JSF and Managed Bean. Input a name, server responds a greeting with that name.

  • Guess-Number: To see if the user input number is equal to the server generated random number.

  • Composite-Component-Example: This is an example for the JSF Composite Component features. The app itself is simple, it tells you if your input number is a prime.

  • Faces-Flows-Example: This is an example for the JSF Faces Flows features. The example module simulates a book shop checkout procedure and club/newsletter subscription procedure. Click here to see A known issue about the warning message in this example

  • Reservation: This simple ticket purchasing app is an example for HTML5-friendly markup, pass-through attributes, and pass-through elements.
